# BOCADITOS - Red Social para entuciastas de la comida

Una red social de prueba para comelones de todo el mundo para compartir sus recetas favoritas y alguna imagen de su plato

## Backend - REST API

- Express
- Mongoose
- Express Async Handler
- DotEnv
- Colors
- Bcrypt
- JsonWebToken

## Frontend

- React (Vite)
- Redux Toolkit
- Styled Components
- React Icons
- React Toastify
- Axios
- React Router Dom
